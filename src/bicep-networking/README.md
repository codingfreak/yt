# Youtube Playlist "Bicep Networking"

[Playlist "Bicep Networking"](https://www.youtube.com/playlist?list=PL6oq24qjix9V3Lou6s9fKpmY-_sTFzjhV)

The goal of the YT series is to show how to use Bicep and related tools to define network resources in Azure. Beginning with basic topics like network design I slowly want to move towards more complicated topics like gateways, VPNs, DNS and the like.
